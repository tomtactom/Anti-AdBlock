# Anti AdBlock
> Ein JavaScript Tool, welches erkennt, ob der Nutzer seinen AdBlocker angeschaltet hat und ihn freundlich darum bittet, diesen auszuschalten.

![](https://repository-images.githubusercontent.com/306378430/ae2b1200-148b-11eb-9aa2-b9d8f4df4572)

Du besitzt eine Webseite und schaltest Werbeanzeigen, doch all deine Nutzer haben einen Adblocker? Dieses Tool hilft dir zuverlässig dabei, freundlich zu erklären, wie der Nutzer diesen ausschalten kann. Wenn der Nutzer dies nicht machen möchte, kann er die Webseite trotzdem aufrufen, muss nur jedes Mal die Meldung, dass der Adblocker ausgeschaltet werden soll wegklicken.

## Installation
Klone dir das Repository und baue es für dich so um, wie es für deine Webseite am besten aussieht.
Du kannst die Farben ggf. an dein Design anpassen.

## Anwendungsbeispiele

* Du besitzt eine Webseite (z. B. einen Blog) auf der du Werbeanzeigen veröffentlichst.
* Du möchtest, dass die Nutzer die Werbung sehen, damit du Geld verdienst
* Du möchtest nicht zu aufdringlich sein und den Nutzern trotzdem erlauben deine Webseite zu besuchen, auch wenn sie einen AdBlocker aktiviert haben.
